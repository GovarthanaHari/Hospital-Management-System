# 🏥 Hospital Management System

A simple **Hospital Management System Website** built using **HTML and CSS**.
This project helps manage hospital-related activities like patients, staff, appointments, and pharmacy.

---

## 🚀 Features

* 📊 Dashboard Overview
* 👨‍⚕️ Patient Management
* 🧑‍⚕️ Staff Management
* 📅 Appointment Booking
* 💊 Pharmacy Section
* 🛏️ Bed Availability

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html`
4. Right-click → **Open with Live Server**
